# Sec-Utils

A minimal offensive security toolkit providing essential CLI utilities for reconnaissance and penetration testing.  
Lightweight, modular, and built for learners and practitioners.

## Features

- **Async Port Scanner** — Fast, concurrent scanning using asyncio.
- **Directory Brute Forcer** — Wordlist-based discovery of hidden routes.
- **HTTP Fuzzer** — Parameter and header fuzzing for basic vulnerability discovery.
- **Hash Identifier** — Detects common hash formats.
- **Wordlist Mangler** — Generates username/password variations.
